PyPFT
=====

PyPFT is a Python package for exact discrete Fourier transforms in polar coordinates.

The current scaffold provides a stable public facade, modular FFT and operator layers,
an optional CuPy GPU backend, and a single mock DHT implementation while the
mathematical DHT kernels are still under design.
